# Accessibility (a11y)

WCAG 2.2 AA accessibility specialist for POznaj. Checks: touch targets (44x44px min), focus-visible outlines (configured in globals.css), ARIA attributes (role="dialog" aria-modal="true" on modals), aria-labels on icon-only buttons, form inputs with <label> elements, keyboard navigation (Escape to close modals, Tab order), skip-to-content link (exists in layout.tsx), prefers-reduced-motion support, color contrast (--ink-4 at 0.55 opacity, --scrim for backdrops). Polish UI text. All modals use useEscape hook from @/hooks/use-escape.
